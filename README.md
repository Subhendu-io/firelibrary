# firelibrary #
eBook management app with React, Node.js, and SQLite.

Steps to run this application.

1. Installing the application node_modules.
```
npm install
```

2. Running the application
```
npm run start
```

3. Open the browser and go to the link bellow
```
http://localhost:5000
```
